# Data Sources

## Pro Football Reference
- Weekly player stats
- Offensive & defensive tables
- Source: https://www.pro-football-reference.com
- Access method: HTML scraping (BeautifulSoup / pandas)

## nflfastR
- Advanced team metrics
- Source: https://github.com/nflverse/nflfastR
- Access method: CSV download

## Identifier Strategy
- Team abbreviations normalized
- Player IDs derived from name + birth year
